Time Assistant

An 80/20-style time coach for Claude Code — classifies your calendar events as high- or low-value, scores your days and weeks, and helps you plan toward an 80% HVT target.

Download for macOS Download for Windows

On Linux, or prefer the terminal? Run curl -fsSL https://claude.ai/install.sh | bash

Note: Claude Code requires a paid Claude plan (Pro, Max, Team, or Enterprise) — the free Claude.ai plan doesn't include Claude Code.

Getting started — 3 steps

  1. 1
    Install Claude Code and sign in
    Download Claude Code from the button above. Follow the installer, then run claude in your terminal and sign in with your Anthropic account.
  2. 2
    Add the marketplace, install, and activate
    Paste these into Claude Code (one at a time). The last one, /reload-plugins, activates the freshly installed plugin in your current session: 
    /plugin marketplace add rimaslogic/time-assistant
/plugin install time-assistant@rimaslogic-plugins
/reload-plugins
    Python is set up automatically the first time Claude Code starts after installation — no manual Python install needed.
  3. 3
    Run the setup wizard
    Type this in Claude Code:
    set up my time assistant
    The wizard connects your Google Calendar, stores an Oura or Timeular token, and renders your first time brief.
